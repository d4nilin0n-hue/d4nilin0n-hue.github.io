    Token finder

    After generating the token on your phone, download the token.txt file and copy it to your PS3 using one of these two methods:

    • Internal HDD (recommended – no USB needed)
      Place the file at:
      /dev_hdd0/tmp/token.txt
    • USB drive
      Insert a USB drive in the PS3 and place the file at the root as:
      /dev_usb000/token.txt
